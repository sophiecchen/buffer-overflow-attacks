# Buffer Overflow Attacks

By Sophie Chen

This is a series of buffer overflow attacks, written in Python, conducted against vulnerable C programs:
* `/exercise1`: Overwrite return address 
* `/exercise2`: Redirect control to shellcode with jump esp trick
* `/exercise3`: Bypass static canary with brute force
* `/exercise4`: Use return-to-libc
* `/exercise5`: Bypass ASLR with nop sled

A full writeup can be found in `writeup.pdf`.
