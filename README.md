# Buffer Overflow Attacks

This is a series of buffer overflow attacks, written in Python, conducted against vulnerable C programs:
* `/exercise1`: Overwrite a variable on the stack
* `/exercise2`: Overwrite a return value directly
* `/exercise3`: Redirect control to shellcode
* `/exercise4`: Overwrite the return address indirectly
* `/exercise5`: Bypass DEP
* `/exercise6`: Bypass ASLR
* `/exercise7`: Use a ROP-based attack
* `/exercise8`: Use Ghidra to reverse engineer a binary
